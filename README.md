# The Theme

Harmony is a responsive jekyll theme by [gayanvirajith](https://github.com/gayanvirajith). 

- Built for jekyll 2.x
- Supports Google analytics and RSS feeds
- Sass based styles
- Browser support: IE 8+, Chrome, Safari and Firefox 
- Fluidly responsive
